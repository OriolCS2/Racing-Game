# Racing Game

This is a game made by Oriol Capdevila and Víctor Segura for an assigment of Physics 2 at CITM University, Terrassa.

The game is a Obstacle Racing Game against the clock in 3d.

The Main Goal is to beat the Time and finish 2 laps before you run out of time.

Controls:
- W: Go forward.
- S: Little slow down. Go backwards.
- A: Turn Left.
- D: Turn Right.
- SPACE: Complete stop.
